# **App Name**: Devfolio

## Core Features:

- Project Gallery: Dynamic project gallery showcasing software development projects.
- Work Experience Timeline: Detailed work experience timeline to highlight professional history.
- Skills Showcase: Categorized skills breakdown displaying expertise.
- Contact Form: Integrated contact form for recruiters to initiate contact.
- Resume Analysis: AI powered tool which summarizes and extracts skills from uploaded resumes to improve relevance.
- Responsive Design: Responsive design for optimal viewing on various devices.

## Style Guidelines:

- Primary color: Vibrant tech-blue (#42A5F5) for highlighting key elements.
- Background color: Dark charcoal grey (#333333) to create a sleek dark theme.
- Accent color: Light sky-blue (#81D4FA) to complement the tech-blue.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body text.
- Use minimalist icons representing various skills and technologies.
- Grid-based layout for organizing content effectively.
- Subtle transition effects to enhance user experience.
